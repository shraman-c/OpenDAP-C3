# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-14
### Added
- Initial release of OpenDAP-C3 firmware.
- Basic playback controls (Play/Pause, Next, Previous).
- Folder navigation support.
- Volume control logic with continuous hold support.
- ADC button ladder decoding with software debounce and hysteresis.
- State preservation using Preferences API.
- Deep sleep implementation for power saving.
- Debug console over UART.
- Automatic recovery and error handling for DFPlayer Mini communication.
